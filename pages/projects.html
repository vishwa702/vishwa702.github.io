Analysis of Global Happiness Indicators   View Code GitHub

An application that provides a bias-free method to analyze happiness scores via advanced data mining techniques including OPTICS and DBSCAN.

Analyzed data for 156 countries across 8 years to extract the most significant factors and derive a custom metric for public happiness. Additionally, I recommend a combination of factors (see the red arrow in the figure) that would most help a particular country increase their happiness score.

Technologies:
  • Time-Series Analysis
  • Clustering
  • Sci-kit learn
  • NumPy
  • Streamlit
